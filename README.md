The program is a payroll system written in Java. It reads employee information from one file and attendance records from another. Employees can log in to see their details, while payroll staff can compute salaries. The system calculates hours worked by checking time in and time out, multiplies it by the hourly rate, and then applies deductions like SSS, PhilHealth, Pag‑IBIG, and withholding tax. After that, it shows the gross pay, deductions, and net pay.

Tho I did not meet the exact result of the codes yet, especially because the program cannot read the attendance file properly and the hours calculation always comes out as zero, the logic and structure are already in place. The program is able to read the files, process attendance, and compute salaries step by step. With more debugging and adjustments, it can fully automate payroll and produce clear reports for employees and staff.
